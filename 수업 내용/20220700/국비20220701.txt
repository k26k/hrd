자바의 타입

1. 기본타입 = 변수의 실제 값을 변수 메모리에 저장
- 정수 타입 변수
- 실수 타입 변수
- 논리 타입 변수

2. 참조타입 = 변수가 사용하는 메모리의 주소 값을 변수 메모리에 저장
- 배열 타입
- 열거 타입
- 클래스
- 인터페이스

메모리 사용 영역
- 메소드 영역 
    JVM(Java Virtual Machine)이 실행될때 생성되고 모든 스레드가 공유하는 영역
- 스택 영역
    JVM 스택은 메소드를 호출할때마다 프레임을 추가하고 메소드가 종료되면 해당 프레임을 제거한다
    프레임 내부에는 로컬 변수 스택 영역이 있는데 기본 타입 변수와 참조 타입 변수가 추가된다.
- 힙 영역
    객체와 배열이 생성되는 영역
    여기에서 생성된 객체와 배열은 JVM 스택 영역의 변수나 다른 객체의 필드에서 참조한다
    참조하는 변수나 필드가 없다면 가비지 콜렉터가 제거함


상속이란?
- 클래스를 정의할때 이미 구현된 클래스를 상속(inheritance) 받아서 
  속성이나 기능(메서드)이 확장되는 클래스를 구현할 수 있다.


임의의 매개변수

- 매개변수가 배열 
    선언시: 반환객체형 함수명(객체형[]  변수){}
    사용시: 함수명(new 객체형[] {변수,  변수}){}, 반환객체형 함수명(배열){}

- 가변 매개변수
    선언시: 반환객체형 함수명(객체형 ... values){}
    사용시: 함수명(변수, 변수, 변수){}
