중첩 클래스(내부 클래스)
- 클래스 내부에 메소드처럼 다른 클래스를 선언해둠

종류
- 인스턴스 멤버 클래스
  class A{
      class B{…}
  }
  A 객체를 생성해야만 사용할 수 있는 B 내부 클래스

- 정적 멤버 클래스
  class A{
      static class B{…}
  }
  A 클래스로 바로 접근할 수 있는 B 내부 클래스

- 로컬 클래스
  class A{
      void method(){
          class B{…}
      }
  }
  method()를 실행할 때만 사용할 수 있는 B 중첩 클래스
