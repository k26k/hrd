

표 여백 종류
바깥 여백 - margin
중간 여백 - border
안쪽 여백 - padding

뱡향은 top, right, bottom, left 4종류

rgb(255, 0, 0) = red
rgb(x, x, x, n) n = alpha, 불투명도 0(투명)~1(불투명)

텍스트에 그림자 효과
text-shadow: (가로) (세로) (번짐) (색상);


박스 모델(box model)
기본 속성 - 너비(wight), 높이(height)
div-width, height

개체 = 객체 = 사물
도형, 이미지, 영상, 텍스트

여백 레이아웃(배치)

img 태그는 옆으로 배치
div 태그는 아래로 배치
마진 중첩 - 마진이 중첩되는 부분은 마진값중 큰거로 적용
테두리(border) = 테두리 기준
바깥-마진, 안쪽-패딩
border-radius = 모서리를 둥글게

패딩(padding) = 콘텐츠 영역과 테두리 사이의 공간
