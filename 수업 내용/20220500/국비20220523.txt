형변환(type conversion)

문자형 -> 숫자형
- Number(문자) -> 정수, 실수
- parseInt(문자) -> 정수
- parseFloat(문자) -> 실수

숫자형 -> 문자형
- toString(숫자) -> 문자

변수(variable)
- val, let (자료형 예약어)

상수(constant variable)
- const (상수형 예약어)

에러 2종류
- 런타임 에러: 실행 중에 발생한 에러
- 컴파일 에러: 컴파일 중에 발생한 에러



제어문

조건문(if문, switch문)

if (조건식){
    조건식이 참일시 실행할 명령어
}

if (조건식){
    조건식이 참일시 실행할 명령어
}else{
    조건식이 거짓일시 실행할 명령어
}

if (조건식1){
    조건식1이 참일시 실행할 명령어
}else if(조건식1){
    조건식1이 거짓이고 조건식2가 참일시 실행할 명령어
}else
    조건식1과 조건식2가 거짓일시 실행할 명령어
}

반복문(while문, for문)

let 초기 값
while(조건식){
    조건식을 만족하는 동안 반복해서 실행할 명령어
    증감값
}

for(초기값; 조건식; 증감값){
    조건식을 만족하는 동안 반복해서 실행할 명령어
}
